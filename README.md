## Quick Start (Local Development)

### Prerequisites
- Node.js 18+ installed
- npm or yarn

### Installation

```bash
# 1. Navigate to project folder
cd portfolio-project

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev
```

The site will open at `http://localhost:5173`
